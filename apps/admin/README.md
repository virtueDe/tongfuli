# Admin App

Tongfuli 的后台管理端，对应 `T022`，当前已推进到第一组可操作工作台。

## 当前状态

- 已把后台首页替换成治理工作台单页。
- 已接通 `T019` 首版接口：剧本导入、来源创建、审核列表与决策、知识发布、策略创建、灰度发布、问答诊断。
- 仍未接入真实鉴权、筛选查询、回滚发布和多页面导航。

## 当前入口

- 首页工作台：`app/page.tsx`
- 前端 API 客户端：`app/_lib/admin-client.ts`

## 启动

```powershell
pnpm --filter admin dev
```

默认端口为 `3100`。

如需指定后端地址，可设置：

```powershell
$env:NEXT_PUBLIC_CORE_PLATFORM_BASE_URL = "http://127.0.0.1:8080"
```

## 验证

```powershell
pnpm --filter admin build
```
