# UI Spec

## 摘要

- 首页为单一主对话入口，不以角色广场作为默认首屏。
- 角色切换以抽屉或侧边浮层呈现，切换后需明确当前发言角色。
- 模式切换保持可见但不抢主路径，默认进入强设定约束模式。
- Web 注重深度体验与专题承接，小程序注重分享回流与轻量访问。

## Web 主站

- 主对话页
- 角色切换抽屉
- 模式切换入口
- 依据展开区
- 热门问题与专题入口
- 分享落地页

## Admin 后台

- 内容资产工作台
- 来源治理工作台
- AI 策略工作台
- 问答诊断工作台
- 风控与审核工作台

## Miniapp

- 轻量主对话页
- 热门角色与热门梗快捷入口
- 分享落地与回流承接
